# 🛡️ Safe Space — Anonymous Workplace Reporting System

> A compassionate, secure, and anonymous platform for reporting workplace concerns with confidence.

---

## 📌 Overview

**Safe Space** is a modern web-based platform designed to empower employees to **safely and anonymously report workplace issues** such as harassment, discrimination, bullying, and ethical violations.

The system prioritizes:
- 🔒 Privacy  
- 🤝 Fairness  
- ⚡ Timely resolution  
- ❤️ Emotional safety  

---

## ✨ Features

- 🔐 **100% Anonymous Reporting**  
  Advanced encryption ensures your identity is never revealed.

- 💬 **Easy Story Submission**  
  Share experiences in a simple, guided form.

- 📄 **Evidence Upload Support**  
  Attach documents, screenshots, or files securely.

- 🧠 **Ethics Committee Review**  
  Reports are evaluated by 5–8 trained members for fairness.

- ⏱️ **Timely Response (Within 7 Days)**  
  No more waiting indefinitely for action.

- 🛡️ **Misuse Prevention System**  
  Detects false reporting and ensures platform integrity.

- 📊 **Pattern Detection**  
  Identifies repeated issues for proactive action.

- 🎨 **Modern UI/UX**  
  Clean, responsive, and user-friendly interface using Bootstrap.

## 🚀 How It Works

1. 👤 User accesses the platform securely  
2. 🔐 Identity is automatically protected  
3. 📝 User submits a report with optional evidence  
4. 🧠 Ethics committee reviews the case  
5. 🔍 System checks patterns & validates information  
6. ✅ Decision is made within 7 days  
7. 📢 Action is taken while keeping anonymity intact
